# 截图目录

此目录用于存放应用程序的截图。

## 文件命名规范

- `main-window_windows.png` - Windows平台主界面截图
- `main-window_macos.png` - macOS平台主界面截图  
- `main-window_linux.png` - Linux平台主界面截图
- `device-list.png` - 设备列表界面
- `device-info.png` - 设备详细信息界面
- `settings.png` - 设置界面
- `connection-status.png` - 连接状态界面

## 截图要求

- 分辨率：推荐 1920x1080 或 1280x720
- 格式：PNG（推荐）或 JPEG
- 包含完整的窗口边框
- 展示核心功能和用户界面
